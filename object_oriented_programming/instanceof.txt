###########################
BINARY OPERATOR: INSTANCEOF
###########################

To check whether an object is the creation of a specific constructor function you can use the binary operator INSTANCEOF. 

The evaluation of the expression will return a bolean true or false.

An instance is an object created by a class, and in JS classes are mimiced by a constructor function.

Syntax:  	

	<object> instanceof ConstructorFunction

	car1 instanceof Vehicle // -> true

Use case can be the if statement.
